# Meme Templates

This folder is for storing meme template images.

## How to Add Templates

1. Download or save meme template images to this folder
2. Name them according to the template names in `script.js`:
   - `drake.jpg` - Drake meme template
   - `distracted.jpg` - Distracted Boyfriend template
   - `buttons.jpg` - Two Buttons template
   - `mind.jpg` - Change My Mind template

## Where to Find Templates

You can find popular meme templates at:
- https://imgflip.com/memetemplates
- https://knowyourmeme.com/
- Or use any image you like!

## Supported Formats

- JPG/JPEG
- PNG
- GIF

## Note

If template images are not found, they will be hidden from the gallery. You can still upload your own images using the "Upload Image" button.
